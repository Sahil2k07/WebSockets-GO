## Try for yourself

1. Clone the repository

   ```bash
   git clone https://github.com/Sahil2k07/WebSockets-GO.git
   ```

2. Change directory

   ```bash
   cd WebSockets-GO
   ```

3. Download all the packages required

   ```bash
   go mod download
   ```

4. Run the project

   ```bash
   go run .
   ```
